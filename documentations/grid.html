Grid


grid 속성은 차트의 격자를 설정합니다.

        options: {
          plugins: {
            xGrid: {
              display: true,
              color: "#323233",
              dash: "10,3",
              weight: 1,
              opacity: .5
            },
            yGrid: {
              display: true,
              color: "#323233",
              dash: "10,3",
              weight: 1,
              opacity: .5
            }
          }
        }

xGrid, yGrid에는 총 5개의 속성이 있습니다.

속성 역할 required default 입력
display 표시 여부 false true true, false
color grid 색깔 false "black" "#226262", "rgb(10, 50, 70)"
dash grid 점선 설정 false "1,0" "int, int" ex)"10,3"
weight grid 두께 false 1 0보다 큰 숫자
opacity right margin false 1 0~1


notify


display

display를 false로 설정하여 grid값만 설정해 놓고 차트에 표시하지 않을 수 있습니다.

설정된 grid값으로 menu 기능으로 사용자에게 grid를 그려줄 수 있습니다.


dash

점선의 "int1,int2"는 int1만큼 선을 그려주고, int2만큼 비워두는 것을 반복합니다.

예를 들어 입력값이 "10,3"일 경우 10,3,10,3,10,3과 같이 선 중간중간에 3의 공백이 들어갑니다.
