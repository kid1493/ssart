Title


title은 총 3가지로 나뉘며 xTitle, yTitle, 그리고 차트 전체 title이 있습니다.

        options: {
          plugins: {
            title: {
              display: true,
              text: 'This is Main Title',
              size: "5px",
              color: 'black',
              align: 'center' //start, end, center
            },
            xTitle: {
              display: true,
              text: 'name',
              size: "5px",
              color: 'black',
              align: 'center' //start, end, center
            },
            yTitle: {
              display: true,
              position: 'left', // left right
              rotate: true, //false일 경우 세로방향
              text: 'value',
              size: "5px",
              color: 'black',
              align: 'center' //top, center, bottom
            },
          }

title과, xTitle의 속성은 동일합니다.

속성 역할 값
display 표시 여부 true, false
text title 값 str
