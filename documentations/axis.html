Axis


axis는 scales에서 각 axis의 최대, 최소, tick분할을 설정하고,

plugins에서 design요소를 설정합니다.

axis plugin은 차트 영역 내부의 배경 설정합니다.

        const myconfig = {
          ...
          options: {
            plugins: {
              axis: 				
                color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                weight: 1,
                opacity: .5,
                dots: {
                  display: true,
                  color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                  weight: 1,
                  opacity: .5
                },
                xAxis: {
                  color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                  weight: 1,
                  opacity: .5,
                  dots: {
                    display: true,
                    color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                    weight: 1,
                    opacity: .5
                  }
                },
                yAxis: {
                  color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                  weight: 1,
                  opacity: .5,
                  dots: {
                    display: true,
                    color: "#323233", // "rgb(255, 0, 0)" "rgba(255, 0, 0, 0.3)",
                    weight: 1,
                    opacity: .5
                  }
                }
              }
            },
            scales: {
              xAxis: {
                ticks: {
                  min: -20,
                  max: 40
                }
              },
              yAxis: {
                ticks: {
                  min: -20,
                  max: 40
                }
             }
            }
          },
          ...
        }

plugins.axis는 총 6개의 속성이 있습니다.

속성 역할 required default 입력
color x,y축 색깔 false "black" "black", "rgb(55, 55, 55)", "rgba(55, 55, 55, 0.5)"
weight x,y축 두께 false 1 int
opacity x,y축 투명도 false 1 0~1
dots x,y축의 tick 설정 false - 아래 dots 표 참고
xAxis x축 설정 false - 아래 xAxis 표 참고
yAxis y축 설정 false - 아래 yAxis 표 참고

axis.dots, xAxis.dots, yAxis.dots에는 총 4개의 속성이 있습니다.

속성 역할 required default 입력
display 표시 여부 false true true, false
color dots 색깔 false "black" "black", "rgb(55, 55, 55)", "rgba(55, 55, 55, 0.5)"
weight dots 두께 false 1 int
opacity dots 투명도 false 1 0~1

axis.xAxis에는 총 4개의 속성이 있습니다.

속성 역할 required default 입력
color x축 색깔 false "black" "black", "rgb(55, 55, 55)", "rgba(55, 55, 55, 0.5)"
weight x축 두께 false 1 int
opacity x축 투명도 false 1 0~1
dots x축의 tick 설정 false - 위 dots 표 참고

axis.yAxis에는 총 4개의 속성이 있습니다.

속성 역할 required default 입력
color y축 색깔 false "black" "black", "rgb(55, 55, 55)", "rgba(55, 55, 55, 0.5)"
weight y축 두께 false 1 int
opacity y축 투명도 false 1 0~1
dots y축의 tick 설정 false - 위 dots 표 참고


notify


axis에서 x, y축을 동시에 설정한 값보다 xAxis, yAxis의 개별 설정 값이 우선 적용됩니다.



scales.xAxis, scales.yAxis에서는 x, y축의 범위를 설정합니다.

아래와 같이 ticks에 min, max 값을 입력합니다.

        scales: {
          xAxis: {
            ticks: {
              min: -20,
              max: 40
            }
          },
          yAxis: {
            ticks: {
              min: -20,
              max: 40
            }
          }
        }
