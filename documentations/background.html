Background


background plugin은 차트 영역 내부의 배경 설정합니다.

        const myconfig = {
          ...
          options: {
            plugins: {
              background: {
                display: true,
                color: "black",
                opacity: .1,
                rx: 20,
                ry: 20
              }
            }
          },
          ...
        }

background는 총 5개의 속성이 있습니다.

속성 역할 required default 입력
display 적용 여부 결정 false true true, false
color 배경 색 false "rgb(241, 243, 245)" "black", "rgb(55, 55, 55)", "rgba(55, 55, 55, 0.5)"
opacity 배경 투명도 false "0.8" "0"~"1"
rx border-radius false 0 int
ry border-radius false 0 int
